= Quick Start =

1. Download and install the V-Oauth plugin.
2. Setup your desired authentication providers' API key/secret in the WordPress backend under *Settings > V-Oauth*.
3. Enable the custom login form via *Settings > V-Oauth > Show login form*, or add it anywhere to your site with the [voa_login_form] shortcode.
